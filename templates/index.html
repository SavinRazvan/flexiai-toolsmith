FlexiAI Toolsmith

A Python framework for building AI assistants with structured tool execution.
Supports CLI and web interfaces with real-time streaming. Compatible with OpenAI, Azure OpenAI, DeepSeek, Qwen, and GitHub Azure Inference.

Multi-Channel Architecture

Publish events to multiple output channels: CLI console, Redis Pub/Sub, or Quart web UI with Server-Sent Events (SSE). Channels are configured via environment variables.

Real-Time Streaming

Stream assistant responses incrementally using OpenAI Assistant API streaming. Supports both CLI and web interfaces with Server-Sent Events for browser-based real-time updates.

Built-In Tools

Includes tools for CSV operations, spreadsheet processing (Excel/OpenPyXL), context storage, YouTube search, subscriber management, and security audit operations. Tools are invoked via Assistant API tool calls.
